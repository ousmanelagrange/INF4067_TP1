2. **Compilation et Exécution du Code Java :**

     ```
     javac *.java
     ```
   - Exécutez le client Java avec la commande :
     ```
     java Client
     ```

3. **Structure du Répertoire :**
   - Le dossier `Structuration` contient plusieurs sous-répertoires, chacun correspondant à un patron de structuration.
   - À l'intérieur de chaque sous-répertoire (Bridge, Adapter, Composite, Decorator), vous trouverez trois sous-répertoires :
     - `Exemple` : Contient le code exemple fait en classe.
     - `Exercice` : Contient le code exercice demandé par l'enseignant.
     - `Modèle` : Contient les diagrammes de classe, y compris le modèle générique, celui de l'exemple et celui de l'exercice en format PDF.

4. **Compilation et Exécution des Exemples ou Exercices :**
   - Naviguez vers le sous-répertoire du patron de structuration de votre choix (par exemple, `Bridge/Exemple`).
   - Compilez les fichiers Java avec la commande `javac *.java`.
   - Exécutez le code Java avec la commande `java NomDuFichierPrincipal`.

5. **Consultation des Diagrammes de Classe :**
   - Naviguez vers le sous-répertoire `Modèle` du patron de structuration de votre choix (par exemple, `Bridge/Modèle`).
   - Ouvrez le fichier PDF correspondant pour consulter les diagrammes de classe.

C'est tout ! Vous devriez maintenant être en mesure de compiler et d'exécuter le code Java et ou de consulter les diagrammes de classe.
